# RocketLaptop

## 기술스택

| Back-End          | Front-End  | Tools&ETC |
| ----------------- | ---------- | --------- |
| JAVA 11           | HTML       | STS4      |
| Spring Boot 2.6.4 | AJAX       | Git       |
| MyBatis           | JavaScript | GitHub    |
| Spring Security   | Bootstrap  | VSCode    |
| MySQL             | JSP        | DBeaver   |
